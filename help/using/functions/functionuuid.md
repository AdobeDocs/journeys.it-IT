# uuid {#uuid}

Genera un UUID casuale (Universal Unique IDentifier).

## Categoria

Stringa

## Sintassi funzione

`uuid()`

## Parametri

Questa funzione non richiede parametri.

## Firma e tipo restituito

`uuid()`

Restituisce una stringa.

## Esempio

`uuid()`

Restituisce &quot;79e70b7f-8a85-400b-97a1-9f9826121553&quot;.s
