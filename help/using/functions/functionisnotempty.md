# isNotEmpty {#isNotEmpty}

Restituisce true se la stringa nel parametro non è vuota.

## Categoria

Stringa

## Sintassi della funzione

`isNotEmpty(<parameters>)`

## Parametri

* stringa

## Firma e tipo restituito

`isNotEmpty(<string>)`

Restituisce un valore booleano.

## Esempio

`isNotEmpty("")`

Restituisce false.

`isNotEmpty("hello")`

Restituisce true.
