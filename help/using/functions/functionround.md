# round {#round}

Restituisce il valore intero più vicino all&#39;argomento con i legami arrotondati all&#39;infinito positivo.

## Categoria

Percorsi

## Sintassi della funzione

`round(<parameters>)`

## Parametri

* decimale
* integer

## Firme e tipo restituito

`round(<decimal>)`

`round(<integer>)`

Restituisce un numero intero.

## Esempio

`round(3.14)`

Restituisce 3.

`round(3.54)`

Restituisce 4.

`round(-3.14)`

Restituisce -3.

`round(3)`

Restituisce 3.
