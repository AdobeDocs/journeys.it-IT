# trim {#trim}

Rimuove gli spazi iniziale e finale.

## Categoria

Stringa

## Sintassi delle funzioni

`trim(<parameters>)`

## Parametro

| Parametro | Tipo |
|-----------|------------------|
| string | string |

## Firma e tipo restituito

`trim(<string>)`

Restituisce una stringa.

## Esempio

`trim(" Hello ")`

Restituisce &quot;Hello&quot;.
