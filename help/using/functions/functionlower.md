# Lower {#lower}

Restituisce una versione in lettere minuscole del parametro.

## Categoria

Stringa

## Sintassi delle funzioni

`lower(<parameter>)`

## Parametro

* string

## Firma e tipo restituito

`lower(<string>)`

Restituisce una stringa.

## Esempio

`lower("A")`

Restituisce &quot;a&quot;.
