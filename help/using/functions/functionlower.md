# lower {#lower}

Restituisce una versione in minuscolo del parametro.

## Categoria

Stringa

## Sintassi della funzione

`lower(<parameter>)`

## Parametro

* stringa

## Firma e tipo restituito

`lower(<string>)`

Restituisce una stringa.

## Esempio

`lower("A")`

Restituisce &quot;a&quot;.
