# upper {#upper}

Restituisce una versione in maiuscolo del parametro.

## Categoria

Stringa

## Sintassi della funzione

`upper(<parameters>)`

## Firma e tipo restituito

`upper(<string>)`

Restituisce una stringa.

## Esempio

`upper("b")`

Restituisce &quot;B&quot;.
