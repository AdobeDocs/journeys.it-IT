# length {#length}

Restituisce il numero di caratteri dell&#39;espressione stringa nel parametro .

## Categoria

Stringa

## Sintassi della funzione

`length(<parameters>)`

## Parametro

* string

## Firma e tipo restituito

`length(<string>)`

Restituisce un numero intero.

## Esempio

`length("Hello World")`

Restituisce 11.
